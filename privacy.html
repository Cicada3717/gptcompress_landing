Privacy Policy - GPTCompress

Last Updated: January 19, 2026

Overview

GPTCompress is a conversation compression tool for ChatGPT. We are committed to protecting your privacy and handling your data responsibly.

Data Collection

What We Collect

  • Conversation Messages: We temporarily process the chat messages you submit for compression
  • No Personal Data: We do not collect names, emails, or any personally identifiable information
  • No Chat History: We do not store or log conversation content

How We Use Data

  • Messages are sent to OpenAI's GPT-4o-mini API for compression
  • Processing happens in real-time
  • Data is immediately discarded after compression
  • No conversation data is stored on our servers

Data Storage

We do NOT store:

  • Conversation history
  • User identities
  • Chat logs
  • Personal information

Third-Party Services

OpenAI API

  • We use OpenAI's GPT-4o-mini API to process compression requests
  • OpenAI's own privacy policy applies: https://openai.com/policies/privacy-policy
  • Data sent to OpenAI follows their data retention policies

Railway (Hosting)

  • Our server is hosted on Railway.app
  • No user data is stored in Railway databases
  • Server logs may contain request metadata (timestamps, request IDs) but no conversation content

Data Security

  • All data transmission uses HTTPS encryption
  • Messages are processed in-memory only
  • No persistent storage of user conversations
  • Server follows security best practices

Your Rights

You have the right to:

  • Not use the service if you don't agree with this policy
  • Request information about our data practices (contact us below)

Changes to This Policy

We may update this privacy policy. Changes will be posted with a new "Last Updated" date.
